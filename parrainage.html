Questions fréquentes

La commission est-elle identique pour les deux modalités de paiement ?

Oui. 1 090 × 20 % = 218 € en paiement unique. 272,50 × 20 % = 54,50 € × 4 versements = 218 € total. Le résultat est toujours le même.

Y a-t-il un plafond ou une durée limite ?

Non. Votre code est permanent. Aucun plafond sur le nombre de filleuls. Chaque filleul génère 218 € de commission, sans limite dans le temps.

Comment fonctionne le lien de parrainage ?

Votre lien contient ?ref=VOTRE_CODE. Le navigateur du visiteur mémorise votre code 30 jours. S'il achète dans ce délai, le lien parrain–filleul est enregistré automatiquement.

Configuration technique (admin)

Taux défini dans js/stripe-products.js via commissionRate: 0.20 sur les produits lifetime et lifetime_4x. Le backend (port 8787) calcule les commissions à chaque webhook Stripe.
